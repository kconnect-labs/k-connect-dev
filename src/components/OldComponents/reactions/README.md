# Система реакций (архивная)

Эта папка содержит архивный код системы реакций, который был удален из основного компонента Post.

## Структура

- `hooks/usePostReactions.ts` - хук для управления реакциями поста
- `components/ReactionsButton.tsx` - компонент кнопки реакций
- `components/index.ts` - экспорты компонентов
- `types/index.ts` - типы для системы реакций

## Использование

Если потребуется восстановить систему реакций, можно использовать код из этой папки.

## Удаленные функции

- Добавление/удаление реакций
- Отображение кнопки реакций
- Подсчет реакций
- Анимации реакций
