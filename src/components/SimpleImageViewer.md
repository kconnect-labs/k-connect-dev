# SimpleImageViewer

Легкий и быстрый просмотрщик изображений без внешних зависимостей.

## Особенности

✅ **Легкий** - без тяжелых библиотек  
✅ **Быстрый** - нативный JavaScript  
✅ **Интерактивный** - зум, перетаскивание, навигация  
✅ **Клавиатура** - поддержка горячих клавиш  
✅ **Адаптивный** - работает на всех устройствах  

## Использование

```jsx
import SimpleImageViewer from './SimpleImageViewer';

<SimpleImageViewer
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
  images={['image1.jpg', 'image2.jpg']}
  initialIndex={0}
/>
```

## Пропсы

- `isOpen` (boolean) - открыт ли просмотрщик
- `onClose` (function) - функция закрытия
- `images` (array|string) - массив изображений или одно изображение
- `initialIndex` (number) - начальный индекс изображения

## Управление

### Клавиатура
- `Escape` - закрыть
- `←` / `→` - предыдущее/следующее изображение
- `+` / `-` - зум
- `0` - сброс зума

### Мышь
- `Двойной клик` - зум 2x / сброс
- `Колесо + Ctrl` - зум
- `Перетаскивание` - перемещение при зуме

### Кнопки
- Кнопки навигации (← →)
- Кнопки зума (+ -)
- Кнопка закрытия (×)
- Счетчик изображений

## Технические детали

- Использует React Portal для рендера в body
- Автоматически блокирует скролл body при открытии
- Поддерживает touch события для мобильных устройств
- Оптимизирован для производительности
